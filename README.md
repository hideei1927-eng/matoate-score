# 的当てスコア（東北新幹線）

親子の的当てゲーム向けの、iPhone利用前提PWAです。  
2人対戦の得点入力、セット集計、勝敗判定、新幹線進行演出、ローカル保存に対応しています。

## V1仕様メモ
- プレイヤー: 2人（初期名は `プレイヤー1 / プレイヤー2`、編集可）
- 1セット投球数: 初期値 `10`（変更可）
- セット数: `1〜4`（初期値 `2`）
- 同点時: 最高セット得点で判定し、それでも同点ならサドンデス（1球ずつ交互入力、最低3球）

## ローカル起動手順
1. このフォルダで静的サーバーを起動します（例）。
   ```bash
   python3 -m http.server 8080
   ```
2. ブラウザで `http://localhost:8080` を開きます。
3. `index.html` が表示されれば利用可能です。

## GitHub Pages 公開手順
1. GitHubで新規リポジトリを作成して、このフォルダをpushします。
2. GitHubのリポジトリ設定を開きます。
3. `Settings` > `Pages` を開きます。
4. `Build and deployment` の `Source` を `Deploy from a branch` にします。
5. Branchは `main`（または利用中ブランチ）/ `/ (root)` を選択して保存します。
6. 数分後に公開URL（例: `https://<user>.github.io/<repo>/`）が発行されます。

## iPhoneでホーム画面に追加する手順
1. iPhoneのSafariで公開URLを開きます。
2. 画面下の共有ボタン（□↑）をタップします。
3. `ホーム画面に追加` をタップします。
4. 名前を確認し、`追加` をタップします。
5. ホーム画面のアイコンから起動すると、スタンドアロン表示で使えます。

## 家族へURL共有して同じように追加してもらう手順
1. GitHub Pagesの公開URLを家族へ送ります（LINEやメールなど）。
2. 家族側もSafariでURLを開きます。
3. `共有` > `ホーム画面に追加` を実行してもらいます。
4. 以後はホーム画面アイコンから同じアプリとして利用できます。

## 保存データ（localStorage）
- 設定値（投球数、セット数、プレイヤー名）
- 直近の試合結果
- 自己ベスト総合点（単一プレイヤーの最高総合点）

## 既知の制約
- iOSのPWAキャッシュは更新反映が遅れる場合があります。更新が見えない場合は以下を実施してください。
1. Safariで対象ページを開いた状態で再読み込み。
2. それでも更新されない場合、ホーム画面アプリを一度削除して再追加。
3. Safariの履歴とWebサイトデータを削除後に再アクセス。
- 本実装はVanilla JavaScriptのみで、バックエンドはありません。端末ごとにデータは独立します（同期されません）。
